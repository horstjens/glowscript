pyramid

.. image:: images/pyramid.png

.. py:function:: pyramid(pos=vec(0,0,0), axis=vec(1,0,0), color=color.green)

   :param pos: Position of center. Default <0,0,0>.
   :type pos: vector
   :param axis: Extends from base to point. Default <1,0,0>.
   :type axis: vector
   :param color: Default *color.white*
   :type color: vector
   :param length: Magnitude of *axis*.  As created by default, along x-axis.
   :type length: scalar
   :param height: By default, y-dimension of base.
   :type height: scalar
   :param width: By default, z-dimension of base.
   :type width: scalar
   :param size: Alternative to length, height, width. Default <1,1,1>
   :type size: vector
